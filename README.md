# FATEC-DSM

Repositório com projetos, atividades e estudos desenvolvidos durante o primeiro termo do curso de **Desenvolvimento de Software Multiplataforma (DSM)** da FATEC.

## 📌 Objetivo

Centralizar e organizar os projetos acadêmicos desenvolvidos ao longo da graduação, servindo como:

- Registro de evolução técnica
- Portfólio para oportunidades na área de desenvolvimento
- Base de estudo e revisão

## 🛠️ Tecnologias utilizadas

As tecnologias variam conforme o semestre e disciplina, incluindo:

- Java
- MySQL
- HTML / CSS / JavaScript
- Git e GitHub
- JDBC
- Outras ferramentas conforme o projeto

## 📂 Estrutura do repositório

Cada pasta representa um projeto ou disciplina específica.
